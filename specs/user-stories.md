# SauceDemo User Stories

## Authentication

1. As a valid shopper, I want to log in with valid credentials so that I can browse products.
2. As an invalid user, I want clear login error messages so that I know why login failed.
3. As a locked-out user, I want to be blocked from login so that account restrictions are enforced.
4. As an authenticated user, I want to log out so that my session is safely ended.

## Inventory and Cart

5. As a shopper, I want to see the product catalog so I can decide what to buy.
6. As a shopper, I want to sort products by name and price so I can find items quickly.
7. As a shopper, I want to add and remove products from cart so I can control my order.
8. As a shopper, I want cart badge updates so I can track selected items.

## Checkout

9. As a shopper, I want to proceed through checkout with shipping info so I can complete purchases.
10. As a shopper, I want field-level validation on checkout so incomplete forms are rejected.
11. As a shopper, I want an order confirmation page so I know the purchase succeeded.

## Session and State

12. As a shopper, I want reset app state to clear cart/session data so I can start fresh.
13. As a shopper, I want app behavior to remain stable across browsers so the flow is reliable.
